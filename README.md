# WIP - Coop Chess

This repository contains the code for a cooperative chess variant, built using [chessground](https://www.npmjs.com/package/chessground), [Bootstrap](https://www.npmjs.com/package/bootstrap), and [Lichess PKCE authentication](https://lichess.org/api#section/Introduction/Authentication).

## Run it on your machine

First, run `npm i` to install dependencies. Then, `npm run build` to build or `npm run serve` to start a local server on [localhost:8080](http://localhost:8080).
